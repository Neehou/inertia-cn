# Inertia 中国子公司官网 — 域名申请分析报告

> 日期：2026-07-16 | 提交人：广州易纳 Web 项目组

---

## 一、背景

Inertia 中国子公司官网（广州易纳）第一版已完成开发并上线测试环境。当前部署于 Cloudflare Pages，使用临时域名 `inertia-cn.*.workers.dev`（该域名在中国大陆无法访问）。正式上线需要申请自有域名并绑定。

**项目定位：** 面向海外工程师和国内硬件企业的 B2B 官网，承载品牌展示、能力介绍、案例背书及 AI/GEO 搜索优化。网站同时服务于两个人群：来华寻找制造伙伴的北美客户，以及寻求出海的国内硬件企业。

---

## 二、域名方案对比

### 方案 A：`inertiapd.cn`（推荐 ⭐⭐⭐⭐⭐）

| 维度 | 分析 |
|------|------|
| **品牌一致性** | 与母公司域名 `inertiapd.com` 仅后缀不同，品牌关联一目了然 |
| **现有资产** | 该域名已有 WordPress 旧站运行中，自带 SEO 权重和历史流量。新站上线后直接替换，搜索引擎已收录的页面可 301 跳转 |
| **客户认知** | 海外客户已知 `inertiapd.com`，自然会尝试 `inertiapd.cn`。零认知成本 |
| **备案状态** | .cn 域名需要 ICP 备案。建议以企业名义备案，周期约 15-20 个工作日 |
| **续费** | ~¥35/年（国内注册商） |
| **技术** | DNS 托管至 Cloudflare，享受免费 CDN + DDoS 防护 + SSL |

### 方案 B：`inertia.cn`（备选 ⭐⭐⭐⭐）

| 维度 | 分析 |
|------|------|
| **品牌简洁** | 4 个音节，比 `inertiapd.cn` 更短，更适合微信公众号、名片等场景 |
| **股权关联** | 不直接体现与母公司 `inertiapd.com` 的关联，但可作为子品牌独立运营 |
| **SEO 风险** | 从零开始积累权重，前期搜索引擎排名较低。但配合新站的 GEO 优化，3-6 个月可建立有效索引 |
| **备案** | 同样需要 ICP 备案 |
| **续费** | ~¥35/年 |
| **可用性** | 需查询当前是否已被注册 |

### 方案 C：`inertia-tech.com`（国际方案 ⭐⭐⭐）

| 维度 | 分析 |
|------|------|
| **不需要备案** | .com 域名在国内注册商购买时无需强制备案，可立即上线 |
| **国际形象** | .com 后缀给海外客户更强信任感，符合全球企业形象 |
| **Cloudflare 原生** | 可直接在 Cloudflare Registrar 购买，DNS 零配置 |
| **续费** | ~$10/年 |
| **劣势** | 与母公司 `inertiapd.com` 的域名关联弱于方案 A |

## 三、成本预算

| 项目 | 金额 |
|------|------|
| 域名注册/续费（1 年） | ¥35–70（国内） / $10（国际） |
| ICP 备案（企业） | 免费（自行办理） |
| Cloudflare CDN + DNS | 免费 |
| Cloudflare Pages 部署 | 免费 |
| **首年总成本** | **¥35–70** |

---

## 四、推荐决策

**首选：`inertiapd.cn`（方案 A）**

理由：
1. 与母公司域名体系一致，海外客户零认知成本
2. 复用现有域名 SEO 权重，新站上线即有搜索基础
3. 维护品牌统一性，子公司形象正式、可信
4. 年成本极低（约 ¥35）

**如 `inertiapd.cn` 无法获取：`inertia.cn`（方案 B）**

**如需立即上线（跳过备案）：`inertia-tech.com`（方案 C）**

---

## 五、时间线

| 阶段 | 时长 | 内容 |
|------|------|------|
| 域名购买 | 1 天 | 选注册商 → 购买 → DNS 托管 Cloudflare |
| ICP 备案（.cn） | 15-20 工作日 | 提交企业资料 → 管局审核 → 备案号下发 |
| 域名绑定 | 1 小时 | Cloudflare Pages → Custom Domain → DNS 生效 |
| 全面上线 | — | 备案通过后即可正式对外 |

---

## 六、需老板确认的事项

1. **是否同意域名预算**（约 ¥35–70/年）？
2. **选择哪个方案**：A (inertiapd.cn) / B (inertia.cn) / C (inertia-tech.com)？
3. **备案主体**：以"广州易纳科技有限公司"名义备案，需提供营业执照副本。谁来对接提供？
4. **上线时间预期**：是否需要先上国际域名（方案 C）绕过备案，还是等 .cn 备案完再上线？
