inertiapd.cn 域名接管需求

日期： 2026-07-17
事由： 中国子公司新官网上线，需接管现有域名
优先级： 紧急（阻塞上线）

背景

Inertia 中国子公司（广州易纳）新官网已完成开发，需要上线替换旧站。当前 inertiapd.cn 为旧版 WordPress 网站，新站需绑定同一域名。

我需要什么

一、域名注册商信息（以下任一项即可）

  • 域名在哪个平台购买？（阿里云 / 腾讯云 / 新网 / GoDaddy / 其他）

二、注册商账号登录信息

  • 账号名 / 手机号 / 邮箱
  • 登录密码

三、如果找不到以上信息

  • 域名当时是谁经办的？（外包建站公司 / 前任 IT 同事 / 其他）
  • 注册时使用的邮箱是什么？（whois 查到的注册邮箱可以看到）

后续做什么（不需要你们操作，我来做）

  1. 拿到账号后，将 DNS 托管迁移至 Cloudflare（免费）
  2. 新网站绑定 inertiapd.cn
  3. 旧站内容做 301 跳转保留 SEO 权重
  4. 配置 SSL 证书（自动）

预计时间

步骤 耗时
找回域名账号 不确定（取决于找到谁）
DNS 迁移到 Cloudflare 1 小时
域名绑定 + SSL 1 小时（DNS 全球生效最长 24 小时）
全面上线 拿到账号当天

备注： 整个过程不涉及任何费用。只是把域名从旧 DNS 服务器迁到 Cloudflare，不需要重新购买或续费域名。新站代码已经部署完毕，就等域名指向即可对外访问。
