# 三大风险

**1. 域名是临时的（`inertia-cn.pages.dev`）**
搜索引擎当测试站处理，不给排名 → 需接管 `inertiapd.cn`

**2. 国内打不开**
Cloudflare 在中国大陆被屏蔽，百度搜不到 → 需 ICP 备案

**3. 旧站权重丢失**
旧 WordPress 站的历史搜索排名清零 → 需 301 跳转迁移
